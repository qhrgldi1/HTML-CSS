# 8강 정리

#### float
* float 처리된 요소는 block 속성을 가짐
* 예) a태그에 float 처리하면 inline 속성이 block 속성을 가지게 됨

#### text-indent: -9999px;
* 마크업된 텍스트가 보이지 않음

#### line-height: 80px;
* a태그의 텍스트 위 아래 빈공간을 클릭할 수 있도록 허용

#### 색깔 상속
* 텍스트 : inherit
* block 요소 : currentColor

#### 헤더 상단 메뉴 UI 꾸미기
* position: absolute; 이용
* left: 0; right: 0; top: 0 하면 border처럼 bar가 나타남
