# 2-3강 정리

#### float
* float 적용할 div 태그는 항상 상위에 위치해 있어야 함

#### text-transform: uppercase;
* 텍스트를 대문자로 치환

#### 텍스트를 맨 앞으로 가져오기
.course-creator,
.course-title {
	position: relative;
	z-index: 10;
}

#### img
* img 태그는 inline 요소

## Grid (그리드)
* ul과 li를 이용
* ul 태그에는 class=".l_row"
* li 태그에는 class=".l_col"
